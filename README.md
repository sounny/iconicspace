# ◎ IconicSpace

**A Visual Archive of Space Industry Identity**

> Companion dataset for "Branding the Void: Visual Identity, Institutional Isomorphism, and Placemaking in the Commercial Space Industry" (Sounny-Slitine & Bensalem, forthcoming, *Space Policy*).

## About

IconicSpace catalogs the visual identities of **200 space organizations** worldwide, organized by:

- **6 Typological Families** (Modernist Heritage, Tech Disruptor, Defense Professional, Sustainability Signaler, Luxury Experiential, National Emblem)
- **5 Cultural Counter-Traditions** (Soviet Constructivist, Chinese Cultural Encoding, Indian Bilingual, African Postcolonial, Japanese Monochrome)
- **20+ Countries** across **6 Continents**

## Key Findings

| Metric | Value |
|--------|-------|
| Sans-serif typography | 93% |
| Abstract (non-literal) iconography | 72% |
| Blue as primary color | 35% |
| Include a rocket | <5% |
